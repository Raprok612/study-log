## **🧠 이론 정리**

### **1️⃣ 종합 실전 핵심 패턴**

| 구분 | 주요 개념 | 출제 포인트 |
|------|------------|--------------|
| **JOIN + 서브쿼리** | LEFT JOIN + 상관쿼리 | NULL 처리 + 조건 최적화 |
| **GROUP BY + HAVING** | 다단계 집계 | 조건 순서 구분 |
| **윈도우 함수** | RANK / DENSE_RANK | 중복 순위 구분 |
| **집합 연산자** | UNION / INTERSECT / MINUS | 컬럼 구조 동일성 |
| **NULL 연산** | NVL / COALESCE / ISNULL | DBMS별 차이 |

> ⚙️ 복원기출 5회차는 실제 시험 문제 구조와 가장 유사한 종합 형태.

---

## **🧮 예제 및 실습**

**예제 1️⃣ – JOIN + 서브쿼리 결합**
```sql
SELECT E.ENAME, D.DEPT_NAME, E.SAL
FROM EMP E
LEFT JOIN DEPT D ON E.DEPTNO = D.DEPTNO
WHERE E.SAL > (SELECT AVG(SAL) FROM EMP WHERE DEPTNO = E.DEPTNO);
```

**예제 2️⃣ – RANK / DENSE_RANK 비교**
```sql
SELECT DEPTNO, ENAME, SAL,
       RANK() OVER (PARTITION BY DEPTNO ORDER BY SAL DESC) AS R1,
       DENSE_RANK() OVER (PARTITION BY DEPTNO ORDER BY SAL DESC) AS R2
FROM EMP;
```

**예제 3️⃣ – 집합 연산 실전형**
```sql
SELECT DEPTNO FROM EMP
UNION
SELECT DEPTNO FROM DEPT;
```
